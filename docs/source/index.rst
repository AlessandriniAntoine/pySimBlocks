pySimBlocks Documentation
=========================

pySimBlocks is a block-diagram simulation library with a Python API and a graphical editor.

This documentation starts with a structured API reference for the code in ``pySimBlocks/``.
The existing guides in ``docs/User_Guide`` and ``docs/Developer_Guide`` are kept unchanged
and can be integrated later without changing this API layout.
